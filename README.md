# 📊 Superstore Sales Analysis

A complete Exploratory Data Analysis (EDA) on a real retail dataset of 9,994 orders
to uncover business insights using Python.

## 🔍 Key Insights Found
- **Total Revenue:** $2,297,201 with a **12.5% profit margin**
- **Technology** is the highest revenue-generating category
- **West region** is the most profitable region
- **Tables** sub-category is losing money due to heavy discounting
- Higher discounts strongly correlate with **negative profit** (shown in scatter plot)

## 📈 Visualizations Created
| Chart | Description |
|---|---|
| Category Sales | Horizontal bar chart of sales by category |
| Monthly Trend | Sales trend from 2014 to 2017 |
| Region Profit | Profit comparison across 4 regions |
| Top 10 Products | Best performing products by revenue |
| Discount vs Profit | Scatter plot showing discount impact |

## 🛠️ Tools & Libraries Used
- Python 3
- Pandas — data cleaning & analysis
- NumPy — numerical operations
- Matplotlib & Seaborn — visualizations

## 📂 Dataset
- Source: [Superstore Dataset — Kaggle](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)
- 9,994 rows × 21 columns
- Date range: 2014 to 2017
